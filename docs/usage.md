# Usage

Mealie MCP exposes consolidated domains for recipes, organizers, households, groups, users, shared resources, administration, exploration, and application utilities. Discover the installed action schemas rather than assuming a historical per-operation tool name.

Use `mealie-mcp` for direct MCP hosting or `mealie-agent` for agent delegation. GraphOS clients should use the canonical `mealie-mcp-operations` skill; its specialist workflows cover meal planning, organizer catalog operations, and recipe management. Begin with reads, require approval for mutations, and verify results without tracing recipe or household content.
